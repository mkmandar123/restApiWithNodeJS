This is a REST API you can do following operations with this API:


Get all articles in database


To get all articles in database you need to send a GET request on this url + /articles and you will get all the articles as response


Post a article


To post a article you need to send a POST request to thisurl/articles and add title and content as parameter in your req and a new post with your given title and content will be created.


Delete all articles from database


To delete all the articles from database send DELETE request to thisurl/articles and all articles will be deleted


Get a specific article


To get a specific article just send GET request to thisurl/articles/articleName where articleName is the name of article you want to retrieve.


Delete a specific article


To delete any particular article send DELETE request to thisurl/articles/articleName where articleName is the name of article you want to delete.


Update a specific article

To update any specific article you can either send a PUT request or a PATCH request,if you send a PUT request then you need to define the title as well as content as parameters but if you send a PATCH request then only the parameters you specified in your request will be updated.
